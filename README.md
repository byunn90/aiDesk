# aiDesk
AI-powered helpdesk chatbot 

## Technologies
- React 
- Tailwind 
- shadcn/ui 
- Express 
- Prisma
- PostgreSQL

## Features (MVP)
- Auth (email/password)
- Chat with history
- OpenAI responses
- PostgreSQL storage

- Embeddings + RAG (pgvector)
- Conversation search

- Frontend: React (Vite), Tailwind, shadcn/ui
- Backend: Express, Prisma, PostgreSQL
- AI: OpenAI Chat + Embeddings
